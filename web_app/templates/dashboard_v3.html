数据集划分

有效样本共 2,732 天（已剔除 2017 地震及新冠封闭日）；数据起自 2016-05-01，按时间顺序划分

为何只用 2023 年以来的数据？

九寨沟于 2017 年 8 月发生地震，景区关闭至 2019 年底；2020–2022 年受新冠疫情反复影响，景区多次限流或封闭。这两段时期的客流模式与政策稳定后的正常运营期存在本质差异，纳入训练会引入分布噪声。2023 年起客流恢复常态、旺淡季规律稳定，因此以 2023-06-01 为建模起点，保证训练数据的同质性与代表性。

模型性能分析

基于测试集（2026-01-07 ~ 2026-04-16，100 天）· Crowd Alert 以旺淡季动态阈值计算（旺季 32,800 / 淡季 18,400）

回归误差 MAE 为平均绝对误差（人次/日），NRMSE 为归一化均方根误差，数值越低越好
预警性能 Crowd Alert F1 综合精确率与召回率；Recall 反映超载日被正确预警的比例，对景区管理尤为重要
训练收敛 Loss = MinMax 缩放后的 MSE；Val Loss 持续下降且无明显过拟合，Early Stopping 有效
特征重要性 XGBoost 基于 Gain 指标；旺淡季标志（58.1%）与昨日客流（22.7%）主导预测，气象特征提供辅助修正
